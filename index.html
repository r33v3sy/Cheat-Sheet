HTML

Document Setup

Tag Element Description
<!DOCTYPE html> Document Type Declaration Required as the first line of an HTML document to tell the browser what type of document to expect.
<head> Head Contains metadata and includes things like the title and links to stylesheets.
<Header> Header Represents a container for introductory content or a set of navigational links. You can have several <header> elements in one HTML document. However, <header> cannot be placed within a <footer>, <address> or another <header> element.
<Nav> Navigation links Represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents. Common examples of navigation sections are menus, tables of contents, and indexes.
